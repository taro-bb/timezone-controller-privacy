# Privacy Policy – Timezone Controller

**Last updated: March 26, 2026**

## Data Collection
Timezone Controller does not collect, store, or transmit any personal data.

## Data Storage
Timezone settings configured by the user are stored locally in the browser using Chrome's storage API. This data never leaves your device.

## Third Parties
No data is shared with or sold to any third parties.

## Contact
If you have any questions, please open an issue on this repository.
